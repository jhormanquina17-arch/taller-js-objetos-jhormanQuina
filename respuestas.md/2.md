1. ¿Qué es una función?
es un bloque donde se puede realizar una tarea especifica ya sea un texto o condicionales o booleanos


2. ¿Qué es un return?
es un comando q se usa para acber la linea de codigo por q al leer linea por linea si el codigo ve el return termina la funcion 


3. ¿Qué pasa si una función no tiene return?
se coree el codigo normal pero al momento de ver en consola sale undefined
